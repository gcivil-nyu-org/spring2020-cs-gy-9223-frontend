CAN Decoder expand_more expand_less

CAN Decoder accepted inputs:
- Base-2 CAN Messages (Numeric & String)
- Base-10 CAN Messages (Numeric & String)

CAN Decoder returns a decoded message in a JSON response structured according to CAN protocol. Returned values are in Base-10, except the "data_bin" and "data_word_x" fields.

Note: For demo purposes, the decoder writes to the database. This feature will be disabled in a future release.
